## Conversion notes

- One recommendation may be split into several changes.
- One change may address several recommendations.
- Add every linked ID in `Converted changes` and in each change proposal’s `Source recommendations`.
- Keep status `Converted` while linked changes are incomplete or the recommendation is only partially addressed.
- Use `Completed` only when the intended outcome is fully delivered or explicitly closed with accepted limitations.
- Do not place executable tasks here.
