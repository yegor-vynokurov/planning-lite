# Quick changes and drift reviews

`QUICK_CHANGES.md` is a compact ledger for tiny edits performed without a full change folder.

A ledger entry is not a substitute for a change when scope or risk is material. It exists so small corrections remain visible and can be periodically compared with the active plan and project documentation.

Run a drift review using `prompts/11-drift-sync.md` when configured thresholds or immediate triggers are reached. Store reports in `reviews/`.
